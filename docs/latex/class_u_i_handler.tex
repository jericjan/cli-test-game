\doxysubsection{Detailed Description}
Handles UI in the game. 

Basically runs the UI in a loop, and the UIs can return other UIs to switch to. Ends when the current UI is a \doxylink{class_game_over}{Game\+Over} screen or when \doxylink{class_user_interface_aa95ee5089a1de7ad211fb3a1bb786e11}{User\+Interface\+::quit\+Game} is set to true.
